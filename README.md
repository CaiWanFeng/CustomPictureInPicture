### 核心难点和关键思路：


#### 1. 如何添加自定义 view 到画中画窗口？
#### 2. 如何隐藏系统的快进快退按钮、播放按钮、进度条？
#### 3. 如何用代码动态修改窗口的形状？
#### 4. 如何用代码旋转窗口？
#### 5. 如何在录视频时不暗屏？
#### 6. 如何让画中画在后台一直运行？
#### 7. 如何应对苹果的审核？
